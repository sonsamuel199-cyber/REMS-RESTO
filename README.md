# REMS RESTO - POS and Inventory System

A web-based Point-of-Sale (POS) and inventory management system developed to optimize business operations.

## Features
- **User Authentication:** Secure login for system access.
- **Inventory Tracking:** Real-time monitoring of stock levels.
- **Sales Management:** Efficient processing of customer transactions.
- **Data Reporting:** Built-in reporting logic for business insights.

## Tech Stack
- **Backend:** PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript

## Usage
1. Clone this repository.
2. Import the database SQL file into your local server (XAMPP).
3. Configure the database connection in `config.php` (if applicable).
4. Run the project in your local browser.
